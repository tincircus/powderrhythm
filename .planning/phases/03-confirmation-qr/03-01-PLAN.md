<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| browser → GET /ticket/:uuid | UUID from URL params — untrusted string passed to Knex WHERE |
| EJS render | ticket.buyer_name from DB rendered in template — must use auto-escape |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-01 | Information Disclosure | GET /ticket/:uuid | accept | UUID v4 = 122-bit entropy; enumeration infeasible at 50-seat scale. Rate limiting deferred to Phase 6 (SEC-04). |
| T-03-02 | Tampering | ticket.ejs EJS output | mitigate | Use <%= %> (auto-escape) for all ticket fields — never <%-. Enforced by task action and acceptance criteria. |
| T-03-03 | Spoofing | APP_URL env var in QR URL | accept | APP_URL is operator-set env var; not user-supplied. No SSRF vector. |
| T-03-04 | Information Disclosure | GET /ticket/:uuid non-confirmed redirect | mitigate | Route queries WHERE uuid AND status='confirmed'; non-confirmed or unknown UUID redirects to / — no ticket data exposed. |
| T-03-SC | Tampering | npm install qrcode | mitigate | slopcheck unavailable — blocking human checkpoint (Task 1) required before install. Verify on npmjs.com/package/qrcode before proceeding. |
</threat_model>
